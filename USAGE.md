# rate-limiter

Token bucket rate limiting middleware

## Installation

```bash
npm install
```

## Usage

```javascript
const rateLimiter = require('./rate-limiter');

// Create an instance
const instance = new RateLimiter();

// Example usage
console.log(instance);
```
